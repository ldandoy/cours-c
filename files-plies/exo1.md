# Exercice court — Inverser une file avec une pile

## Énoncé

### Écrire un programme qui :

- remplit une file avec plusieurs entiers,
- transfère tous les éléments dans une pile,
- retransfère les éléments dans une nouvelle file,
- affiche le résultat.

## But

Montrer que la pile permet d’inverser l’ordre.

## Exemple :

File initiale : 1 2 3 4
Pile : 4 3 2 1
File finale : 4 3 2 1
